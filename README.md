# slack-clone-app

This is a Slack clone application built with ReactJS, Material UI, and Firebase

## How to use

Clone the repo and use npm to install dependencies.

```bash
npm install
```

Visit: https://slack-clone-app-6da0b.web.app/room/7eniEFMviUhEEWFRQ5ag
